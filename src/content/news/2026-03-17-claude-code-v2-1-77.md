# Claude Code v2.1.77 更新内容

Claude Opus 4.6とSonnet 4.6のトークン制限を拡大（最大64k、上限128k）し、サンドボックスファイルシステムに`allowRead`設定を追加、`/copy`コマンドのインデックス指定対応などの機能拡張と、複合bashコマンドの権限設定、自動更新時のメモリ蓄積、会話履歴の切詰め、PreToolUseフック、ファイル行末コード、長時間実行時のメモリ増加、APIフォールバック時の使用量追跡など複数の不具合を修正しました。
