# v2.1.85 要約

MCP サーバーの複数対応、フック条件付け実行、スケジュールタスクのタイムスタンプ表示、OAuth RFC 9728 対応、組織ポリシー準拠など複数の機能追加と、コンテキスト超過時の `/compact` 失敗やコンピュータ使用ツールのマルチモニタ対応など15件以上のバグ修正が含まれています。
